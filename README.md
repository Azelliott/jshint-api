# JSHint GIUI

## About 
Graphical interface to JSHint API provided by Code Institute

The API provides multiple ways and options to validate JS files:


* filename

* url

* options
    * ES6
    * ES8
    * Harsh
    * jQuery
    * Relax
    * Strict

* code

NOTE: this API key expires on 24-07-2023.
